Subject: File No. S7-2026-15
From: Michael Castleberry

As an active investor, I would prefer companies continue to submit quaterly earnings report.

As a former CAO, it is my opinion that the traditional 10-Q could be greatly streamlined as follows:

Eliminate note on Significant Accounting Policy, expect for changes adopted since the prior 10-K / 10-Q.

Fair Value Measurement - eliminate all the boiler plate definition - refer reader to most current 10-K.

Committments and Contingencies - refer reader to most recent 10-K and only discuss changes that ocurred during the current quarter.

Controls and Procedures - eliminate need to say anything if there have been no changes since the last 10-K
